What is Fronty?

A web framework.

Fetures:

  1. Easy to use
  2. Simple
  3. Lightweight
  4. Fast

Easy to use

Fronty is a simple, lightweight, and fast website framework written in Python.
It is designed to be easy to use and easy to learn. It is also designed to be easy to extend and customize.
Fronty is a great choice for beginners and experts alike.
Fronty is also a great choice for people who want to learn how to make websites.
Fronty is also a great choice for people who want to learn how to make websites with Python and without knowing any kind of web technologies like HTML, CSS, and JavaScript.
